# cvQTcameraGL
- Project using OpenGl,QT,OpenCV and GLSL Shaders
- WIP

# Build
- `mkdir build && cd build`
- `cmake ..`
- `make`

# usage
- run `./cvQTcameraGL`
- enter id of cam (e.g. 0,1 etc)
- enter folder of shader: 
  - `shader_AA`  
  - `shader_AB`  
  - `shader_BA` 
  - `shader_BB`
- shader not working yet
